# Simon Says Game

A web-based Simon Says memory game built with HTML, CSS, and JavaScript.

## Description
This project recreates the classic Simon Says game. The game displays a sequence of colored pads, and the player must repeat the sequence correctly. Each round adds one more color to the pattern. If the player clicks the wrong color, the game ends and resets.

## Features
- Four interactive colored pads
- Random sequence generation
- Round counter
- High score tracker
- Reset button
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Click the **Start Game** button.
2. Watch the sequence of flashing colors.
3. Repeat the sequence by clicking the pads in the same order.
4. Continue playing as the sequence gets longer each round.
5. If you click the wrong color, the game ends.
